# Ritual - Web3 Payment Platform

## Features

- MetaMask Wallet Integration
- Ritual Testnet Support
- Wallet Auto Connect
- Balance Display
- Send Transactions
- Transaction History
- Responsive Dashboard
- Modern Web3 UI
- React Router Navigation
- Ethers.js Integration

  ## Tech Stack

- React.js
- Vite
- JavaScript
- Ethers.js
- React Router DOM
- React Icons
- CSS3
- MetaMask
- Ritual Testnet

  Developed Ritual, a Web3 payment platform using React, Vite, and Ethers.js with MetaMask integration. Implemented wallet connection, Ritual Testnet switching, balance tracking, transaction handling, and a responsive dashboard interface for decentralized payments.
